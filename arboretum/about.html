TreeScript Quickstart

Live Demo

TreeScript supports integers, booleans and all of the operations one would expect over these types (+, -, *, &&, ||, etc). TreeScript programs are a series of assignment statements, for instance 'a = 1 + 2'

TreeScript also supports functions, in the form of lambdas. Function definition: 'id = \x -> x'. Function application: 'result = id 1'
